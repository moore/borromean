# Archived Core Specification Pilot

Status: superseded design exploration; not current design authority.

This directory preserves the first attempted decomposition of the replacement
Borromean core specification. It contains valuable design ideas and verification
questions, but its principal value was informing the active system narrative and
incremental review process.

Active work lives in:

- [the system narrative](../../core/000-system-narrative.md);
- [the incremental design queue](../../core/todo.md); and
- [the authoring guide](../../core/authoring.md).

The archived draft begins with the [pilot index](pilot-index.md). Supporting
historical material is grouped by purpose:

- [design and audit context](context/README.md);
- [the executable model pilot](../../../models/archive/core-pilot/README.md);
  and
- [the Rust implementation snapshot](implementation/README.md).

These artifacts are preservation inputs for the active implementation inventory.
They are deliberately outside active specification, model, and compiled-source
namespaces and must not be cited as current conformance evidence.
